# Machine Learning - Stanford University

**Provider:** Stanford University (Coursera)  
**Instructor:** Andrew Ng  
**Course Type:** Complete with exercises

## Course Overview

This is the foundational machine learning course that covers core algorithms and techniques used in the field. The course includes 8 comprehensive programming exercises implemented in MATLAB.

## Target Audience

- Beginners to machine learning
- Students wanting to understand ML fundamentals
- Developers transitioning to ML/AI

## Learning Objectives

- Understand supervised and unsupervised learning
- Implement key ML algorithms from scratch
- Apply ML to real-world problems
- Understand bias/variance tradeoffs

## Course Content

### Exercises (8 total)

1. **Linear Regression** - Implement gradient descent and normal equations
2. **Logistic Regression** - Classification with regularization
3. **Multi-class Classification** - One-vs-all classification
4. **Neural Networks** - Forward and backpropagation
5. **Regularization** - Bias/variance analysis
6. **Support Vector Machines** - SVM implementation
7. **K-means & PCA** - Unsupervised learning
8. **Anomaly Detection & Recommender Systems** - Advanced applications

## Prerequisites

- Basic linear algebra
- Basic programming (MATLAB knowledge helpful but not required)
- Calculus basics

## Content Structure

- `machine-learning-ex1/` through `machine-learning-ex8/` - Exercise directories
- Each exercise contains:
  - Starter code (`.m` files)
  - Data files (`.mat`, `.txt`)
  - Exercise PDF with instructions
  - Submission framework

## How to Use

1. Start with `machine-learning-ex1/` and work through sequentially
2. Read the exercise PDF for each week
3. Complete the required functions
4. Test your implementation
5. Submit (if using Coursera platform)

## Key Topics Covered
